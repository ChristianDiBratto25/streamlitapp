# Company Name Cleaner

A simple web app that helps you clean company names by removing common suffixes like Inc., Co., Corp., etc. from your CSV files.

## Features
- Upload CSV files containing company names
- Automatically removes common business suffixes
- Preview cleaned data before downloading
- Download cleaned data as CSV
- See statistics about changes made

## How to Use
1. Upload your CSV file containing company names
2. Select the column that contains the company names
3. Click "Clean Company Names"
4. Preview the results
5. Download the cleaned data

To run this app locally:

1. Clone this repository

2. Install dependencies
```bash

```

